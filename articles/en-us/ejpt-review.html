Junior Penetration Tester (eJPT) Review

Mar 29, 2026 by Brenno M.

The eJPT, eLearnSecurity Junior Penetration Tester, offered by INE Security, is a practical entry-level certification focused on penetration testing, with emphasis on network enumeration, vulnerability exploitation, Metasploit Framework usage, pivoting, and web application testing.

On 03/29/2026, I took the exam and passed with approximately 90% score.

Topics Covered

The certification covers the following domains and topics:

  • Assessment Methodologies (25%)
    • Network reconnaissance
    • Port and service identification
    • OS Fingerprinting
    • OSINT techniques
  • Host and Network Penetration Testing (35%)
    • Exploit identification and modification
    • Metasploit Framework
    • Network Pivoting
    • Brute-force attacks
    • SMB enumeration
    • Lateral movement
    • Privilege escalation
  • Host and Network Auditing (25%)
    • System enumeration
    • User account gathering
    • Credential extraction
    • File and directory enumeration
  • Web Application Penetration Testing (15%)
    • Web vulnerability identification (SQL Injection, XSS)
    • Hidden resource discovery
    • CMS attacks (WordPress, Drupal)
    • Login panel brute-forcing
Therefore, it is extremely important to master all the topics listed above.

About the Exam

This is a 100% practical and hands-on exam consisting of 45 scenario-based questions. The candidate is placed in a browser-accessible lab environment simulating a corporate network with multiple segments, including a DMZ and an internal network. Questions are answered based on findings obtained through active exploitation of the targets.

The candidate has 48 hours to complete the exam, requiring at least 70% correct answers (32 out of 45) to pass. The exam is open book, allowing the use of notes, cheatsheets, and documentation. There is no proctoring, and it can be taken remotely from anywhere with a stable internet connection. The voucher includes 1 free retake and is valid for 180 days from purchase. The certification is valid for 3 years.
